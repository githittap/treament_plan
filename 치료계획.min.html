🖼️ 이미지 불러오기

  • 파노라마 이미지 업로드 버튼 클릭 또는 화면에 Ctrl+V로 클립보드 이미지 붙여넣기
  • 이미지 삭제: 좌측 "현재 이미지 삭제" 버튼 클릭, 또는 Delete (Ctrl+Z로 복원 가능)

🦷 치료종류 마커 입력

  • 우측 패널에서 치료종류 선택(또는 단축키) → 이미지 클릭 → 팝업에서 치아번호 입력 → Enter
  • 임플란트는 종류(EV/ST/DT/BD)와 시기도 선택 가능 → 버튼 클릭 후 Enter
  • 마커 더블클릭 → 내용 수정
  • 마커 위에서 마우스 휠 → 크기 조절
  • 선택된 마커에서 ESC → 마커 삭제

🗂️ 표시도구 (발치·RCT·GBR 등)

  • 우측 패널 "표시 도구" 섹션에서 항목 선택 → 이미지 클릭 → 즉시 스티커 삽입 (팝업 없음)
  • 발치(1): X 마크 / RCT(V): 빨간 박스 / GBR(G): GBR 스티커
  • 표시(빨간펜)·연결선(초록 점선): 드래그하여 그리기

📝 텍스트 입력 (T)

  • T키 또는 "텍스트" 버튼 → 이미지 클릭 → 내용 입력 (줄바꿈 지원) → Ctrl+Enter로 확인
  • 텍스트 더블클릭 → 수정 팝업
  • 텍스트 위에서 마우스 휠 → 글자 크기 조절

🔲 다중 배치 모드 (R)

  • R키 또는 우측 상단 파란 버튼으로 활성화 (버튼이 파란색으로 변함)
  • 치료종류 선택 → 이미지 여러 곳 클릭하여 점 배치
  • 하단 팝업에서 각 치아번호 입력 → "전체 적용" 클릭 또는 Enter
  • 적용 후 자동 해제됨

🖱️ 선택/이동 모드 (A)

  • 마커 클릭 → 선택 (파란 테두리)
  • 선택 후 드래그 → 위치 이동
  • 모서리 핸들 드래그 → 크기 조절
  • 선택 후 ESC → 삭제

💾 저장 / 공유

  • 이미지 복사 (Ctrl+C): 작업 결과를 클립보드에 복사 → 카카오톡 등에 바로 붙여넣기
  • PNG 다운로드: 파일로 저장

2026-06-22 (최신) BY Mr. jung
  • 단축키 전면 개정: C=코어Cr, V=RCT(스티커), G=GBR, D=완료, H=보류, 1=발치
  • RCT → 표시도구로 이동 (빨간 박스 스티커, 팝업 없음)
  • 발치 → 표시도구 항목으로 이동 (X마크 즉시 삽입)
  • Ctrl+Y: 다시실행(redo) 추가
  • Ctrl+C: 선택 없이도 항상 이미지 복사 가능
  • Delete: 이미지 삭제, Ctrl+Z로 복원 가능
  • 다중 배치 모드 버튼 크게 개선 (파란 버튼, ON시 색상 변경)
  • 마커/글자 크기 슬라이더에서 마우스 휠로 조절 가능
  • 텍스트 줄바꿈 지원 (Ctrl+Enter로 확인)
  • 텍스트 기본 크기 1.3배 확대
  • ESC: 팝업 닫기 / 선택 마커 삭제
  • 치료종류 라벨 배경색 병원 테마색(#156f72)으로 변경
