설명덱 제작기 사용법

코딩 없이, 왼쪽에서 슬라이드를 추가하고 가운데서 내용을 채우면 오른쪽에 발표가 실시간으로 그려집니다. 다 만들면 ‘발표 파일 만들기’로 더블클릭용 파일을 뽑으세요.

3단계

  • 추가 — 왼쪽 ‘＋ 슬라이드 추가’에서 종류 선택 (표지·요약·도표·표·비교·체크리스트)
  • 작성 — 가운데 칸을 타이핑. ‘등장 순서’ 숫자 = 클릭 몇 번째에 나올지 (0 = 열자마자)
  • 내보내기 — ‘발표 파일 만들기’ → 인터넷 없이 더블클릭으로 열리는 발표물 완성

저장 vs 내보내기

  • 저장 — 이 브라우저에 임시 보관(작업 이어하기용). ‘불러오기’로 다시 엶
  • 발표 파일 만들기 — 남에게 보여줄 완성본 HTML을 내려받음
