# 임플란트 기록 생성기 — 설계 문서

> 작성일: 2026-06-24 · 대상 파일: `치료계획.html` (단일 파일 통합)

## 목적

임플란트 식립 기록을 간결한 한 줄 텍스트로 생성하는 도구. 치료계획(파노라마 마커) 기능과는
**별개의 포맷·데이터**이지만 같은 `치료계획.html` 안에 통합한다. 생성된 텍스트는 복사해서
외부(진료차트 등)에 붙여넣는 용도.

출력 예시:
```
2026-06-24 (ST #24 23) > 2026-09-08 ISQ & SCAN. (30N이상)
```

## 진입점

- 별도 "모달 열기" 버튼을 만들지 않는다.
- 상단 초록색 헤더 바(`.clinic-header`, 라인 71~79) 안, 제목 영역과 "STAFF ONLY" 배지
  사이의 빈 공간에 **`🦷 임플란트 기록`** 버튼을 배치한다.
- 클릭 시 캔버스와 분리된 별도 모달 팝업이 열린다.

## 입력 흐름 (단계별 팝업, 한 번에 하나씩)

1. **식립 날짜**
   - 기본값 = 컴퓨터 현재 날짜를 `YYYY-MM-DD`로 자동 채움. 수정 가능. Enter로 확정.
2. **임플란트 종류** (한 개 선택)
   - 버튼: `EV` `ST` `DT` `BD` `EV mini` `ST mini` `BD nc`.
3. **치식 번호**
   - FDI 차트를 그대로 표시하고 셀 클릭으로 입력.
   - 여러 개 클릭 가능. 같은 종류·같은 경과월이면 한 줄에 묶어서 표시.
4. **경과 개월수**
   - 기본 4개만 표시: `2.5M` `3M` `4M` `6M`.
   - **`＋`** 버튼을 누르면 추가 옵션(`5M` `7M` `8M` `9M` 등)이 펼쳐진다.
   - 식립일 + 개월수 → ISQ&SCAN 날짜 자동 계산 (월말 보정 포함).
5. **메모 (선택)**
   - 측정 버튼: `ISQ + SCAN` · `ISQ만 측정` · `경과관찰`.
   - 초기고정 값 버튼: `0N` `5N` `10N` `20N` `30N이상` ("초기고정"이라는 단어 없이 값만).
   - 기타 자유 입력칸.

## 출력 & 복사

- 입력된 기록을 행 목록으로 누적 표시.
- **복사** 버튼:
  - 종류·날짜·경과월·메모가 모두 동일한 항목 → 간결한 한 줄로 병합.
  - 하나라도 다르면 → 행을 나눠 여러 줄로 출력.
  - `navigator.clipboard.writeText`로 클립보드에 복사.

## 데이터 모델

- 치료계획 `annotations`와 무관한 별도 배열 `implantRecords[]`.
- 캔버스에 그리지 않음.
- 저장/불러오기 JSON 포함 여부는 기본 **미포함**(단순 텍스트 생성기). 추후 요청 시 검토.

## 비범위 (YAGNI)

- 캔버스 렌더링·마커 연동 없음.
- 다중 사용자, 서버 저장, 통계 없음.
- 과거 차트 일괄 수정 UI 없음(예외적으로 여러 임플란트가 다른 날짜/종류일 때는 행 분리로 자연 처리).
